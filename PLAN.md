# 纳西文化博物馆网站 — 任务与计划

## 项目概述

参考网站：Leonardo Frigo 手工地球仪网站 (leonardofrigo.com)
设计方向：纳西古韵 × 现代精致，突出东巴象形文字与纳西文化
技术栈：纯 HTML / CSS / JS，无框架依赖
色彩体系：深靛蓝（纳西蓝染布）、朱砂红、玉龙雪白、古城石板灰、古城木色、经文金

## 页面结构（10 个区块）

### 1. 全屏 Hero — "发现千年活着的象形文字"
- 全屏 100vh，背景图 + 暗色遮罩
- 大标题衬线体 + 英文副标题
- 东巴文符号浮动装饰（CSS 动画）
- Ken Burns 背景缩放 + 鼠标视差 + 文字逐字淡入

### 2. 东巴文无限滚动横幅
- 全宽横条，两行方向交替
- 12 个东巴文 SVG 符号水平无限滚动
- 悬停减速

### 3. 博物馆简介 — 分屏布局
- 左 50% 大图 | 右 50% 文字
- 纳西族历史、丽江古城、东巴文化介绍
- 滚动触发左右滑入

### 4. 东巴文字专题（核心亮点）
- 深靛蓝背景，4 列卡片网格
- 每张卡片：东巴文 SVG 符号 + 中文释义 + 拼音 + 英文 + 文化解读
- 例字：太阳、山、水、人
- 卡片逐个弹入 + 悬停旋转发光

### 5. 文化展品画廊
- 不等宽网格（2-3 列），6 张展品卡片
- 东巴经书、纳西古乐、蓝印花布、古城建筑、纳西服饰、东巴画
- 图片悬停缩放 + 遮罩文字 + 灯箱点击放大

### 6. 纳西古乐 — 视听体验区
- 全宽背景 + 居中内容
- 模拟音频波形 SVG 动画 + 播放按钮交互
- 背景视差

### 7. 东巴文字互动体验
- 左 60% Canvas 画布 | 右 40% 模板面板
- 鼠标/触摸绘制，墨水扩散效果
- 6 个东巴文字模板 + 清除/切换笔触

### 8. 文化传承故事 — 图文交替
- 两个交替图文区：东巴祭司传承 / 纳西古乐保护
- 滚动视差 + 文字逐行淡入

### 9. 参观信息 & CTA
- 深靛蓝背景，居中内容
- 地址、开放时间、门票 + "预约参观" 按钮
- 背景东巴文符号浮动 + 按钮发光

### 10. 页脚
- 深色多列布局
- Logo、导航、社交媒体、版权

## 动效清单

### 滚动触发（IntersectionObserver）
| 元素 | 效果 |
|------|------|
| Section 标题 | 从下方 30px 滑入 + 淡入 |
| 图片 | 渐显 + 轻微放大 |
| 卡片 | 逐个弹入（staggered 0.15s） |
| 文字段落 | 从左/右滑入 |

### 悬停效果
| 元素 | 效果 |
|------|------|
| 导航链接 | 底部下划线从中心展开 |
| 东巴文符号 | 旋转 + 阴影发光 |
| 卡片 | 上浮 5px + 阴影增强 |
| 展品图片 | 缩放 1.05 + 遮罩文字上滑 |
| 按钮 | 背景色变化 + 缩放 |

### 页面级效果
- 自定义光标跟随（东巴文符号）
- 平滑滚动
- 页面加载入场动画
- 顶部滚动进度条
